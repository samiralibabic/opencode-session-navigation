# Support

`opencode-session-navigation` is maintained on a best-effort basis.

## Supported Scope

- Latest released version of this plugin
- OpenCode versions matching `engines.opencode` in `package.json`
- Standard OpenCode TUI plugin installation through `tui.json` or `opencode plugin <module>`

## Getting Help

For bugs or feature requests, open a GitHub issue:

https://github.com/samiralibabic/opencode-session-navigation/issues

Please include:

- Plugin version
- OpenCode version
- Operating system and terminal
- Install method: npm, GitHub, or local path
- Relevant `tui.json` plugin config
- Steps to reproduce
- Expected behavior and actual behavior

For security issues, see `SECURITY.md`.
